HTML Tags Dictionary

This reference guide explains the HTML tags used in this website.

Tag Description
<!DOCTYPE html> Declares the document type and HTML version.
<html> The root element that contains all other HTML elements in a document.
<head> Contains meta information, links to stylesheets, scripts, and the title of the document.
<meta> Provides metadata about the HTML document, like character encoding and viewport settings.
<link> Used to link external resources like stylesheets to the HTML document.
<title> Defines the title of the document shown in the browser's title bar or tab.
<body> Contains all the visible content of the webpage.
<aside> Defines content that is tangentially related to the content around it, like a sidebar.
<img> Embeds an image in the HTML document.
<h1> to <h6> Heading elements that represent six levels of section headings, with h1 being the highest level.
<p> Represents a paragraph of text.
<div> A generic container for flow content, which has no semantic meaning by itself.
<a> Creates a hyperlink to other web pages, files, or locations within the same page.
<i> Traditionally used for italic text, but now commonly used for icons with Font Awesome.
<main> Specifies the main content of a document, excluding headers, footers, and sidebars.
<nav> Defines a section that contains navigation links.
<ul> Defines an unordered list of items.
<li> Represents a list item in ordered or unordered lists.
<section> Represents a standalone section of content within a document.
<hr> Creates a thematic break or horizontal line.
<strong> Indicates text with strong importance, typically displayed as bold.
<br> Produces a line break in text.
<table> Defines an HTML table.
<thead> Groups the header content in an HTML table.
<tbody> Groups the body content in an HTML table.
<tr> Defines a row in an HTML table.
<th> Defines a header cell in an HTML table.
<td> Defines a standard data cell in an HTML table.
<footer> Defines a footer for a document or section, typically containing authorship information.

Common HTML Attributes

Attribute Description
class Specifies one or more class names for an element, used with CSS and JavaScript.
id Provides a unique identifier for an element, used for CSS styling and JavaScript manipulation.
src Specifies the URL of the external resource (used with elements like img, script, etc.).
href Specifies the URL of the page the link goes to (used with anchor tags).
alt Provides alternative text for an image if it cannot be displayed.
target Specifies where to open the linked document (e.g., _blank for a new tab).
